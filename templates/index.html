DFA: A Deterministic Finite Automaton is defined as a 5-tuple (Q, Σ, δ, q₀, F) where:

  • Q → Finite set of States
  • Σ → Set of Input Symbols
  • δ → Transition Function
  • q₀ → Initial (Start) State
  • F → Set of Final (Accepting) States

Minimization is required to obtain the minimal version of any DFA, which consists of the minimum number of states possible while recognizing the exact same language.
